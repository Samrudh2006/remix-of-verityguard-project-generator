Getting Started

VerityGuard is an advanced project generator that creates production-ready full-stack applications with comprehensive features including authentication, database integration, CI/CD pipelines, and much more.

Quick Start

  1. Visit the Project Generator
  2. Configure your project settings
  3. Select frontend and backend frameworks
  4. Choose your database
  5. Enable advanced features
  6. Generate and download your project

Installation

npm install -g verityguard-generator
verityguard init my-project
cd my-project
npm install
npm run dev

Backend Development

Choose from multiple backend frameworks with built-in best practices.

Express.js (Node.js)

// Example Express route
import express from 'express';

const router = express.Router();

router.get('/api/users', async (req, res) => {
  const users = await User.findAll();
  res.json(users);
});

export default router;

FastAPI (Python)

# Example FastAPI route
from fastapi import FastAPI

app = FastAPI()

@app.get("/api/users")
async def get_users():
    users = await User.find_all()
    return users

Troubleshooting

Common issues and solutions.

Port Already in Use

ERROR: Port 3000 is already in use

Solution: Change the port in .env file or kill the process:
lsof -ti:3000 | xargs kill -9

Database Connection Failed

ERROR: Could not connect to database

Solution: Check your DATABASE_URL in .env and ensure the database is running:
docker-compose up database

Build Errors

ERROR: Build failed

Solution: Clear cache and rebuild:
rm -rf node_modules dist
npm install
npm run build
